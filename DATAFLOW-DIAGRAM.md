### saveHabits() Function

```javascript
function saveHabits() {
    // 1. Cek apakah user sudah login
    if (!currentUser) return;
    
    // 2. Cari user di array users
    const userIndex = users.findIndex(
        u => u.email === currentUser.email && 
             u.authMethod === currentUser.authMethod
    );
    
    // 3. Jika user ditemukan, update habitsnya
    if (userIndex !== -1) {
        users[userIndex].habits = habits;  // Update array
        
        // 4. Simpan ke localStorage
        localStorage.setItem(
            'neuroHabitUsers', 
            JSON.stringify(users)
        );
    }
}
```

## âš ï¸ Important Notes

1. **localStorage Bukan Backend**: Data hanya tersimpan di browser lokal
   - Jika user buka dari device lain â†’ tidak ada datanya
   - Solusi: Perlu backend/server untuk sinkronisasi

2. **Private/Incognito Mode**: localStorage tidak bekerja
   - Browser tidak menyimpan localStorage di private mode
   - Akan hilang saat tab ditutup

3. **Clear Cache**: Jika user clear browser cache, data akan hilang
   - Tidak ada cara untuk mencegah ini (intentional user action)

4. **Capacity Limit**: localStorage ~5-10MB per domain
   - Untuk aplikasi sedang-besar, gunakan IndexedDB

5. **Security**: localStorage dapat diakses via JavaScript
   - Jangan simpan data sensitif (password, token auth)
   - Gunakan secure cookies untuk auth token
